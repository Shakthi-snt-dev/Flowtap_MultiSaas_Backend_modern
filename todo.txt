How to use
Start everything:

cp .env.example .env          # fill in your secrets
docker-compose up -d
Start one industry only:

docker-compose up -d postgres redis food-api food-ui
Rebuild after code change:

docker-compose build food-api && docker-compose up -d food-api

    -c Release -o APIEndpoints/$api/output/build --no-build &
done
